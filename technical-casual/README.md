# Technical Casual

This folder now contains the first Humanish skill focused on technical writing with a casual, human voice.

The skills that land here should help an agent explain technical ideas clearly, avoid stiff or overproduced phrasing, and sound like a smart person talking to another smart person.

Current skill:

- `technical-casual`: general-purpose technical explainer format built around concrete examples, step-by-step visuals, intuition-building, and clear human-sounding prose
