# Technical Casual

This folder is reserved for the first Humanish skills focused on technical writing with a casual, human voice.

The skills that land here should help an agent explain technical ideas clearly, avoid stiff or overproduced phrasing, and sound like a smart person talking to another smart person.

Add each new skill as its own folder or `SKILL.md`, depending on how you want to organize the collection.

Current skills:

- `kirupa-technical-casual`: Kirupa-style technical explainer format built around a concrete example, step-by-step visuals, rejected alternatives, and real-world applications
